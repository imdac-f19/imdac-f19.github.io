# CSS z-index

z-index allows us to layer our positioned elements on top of eachother. We can put text over a background image, controls over content, and order our elements however we would like. Each of the squares below have a fixed position to the top left of the screen. Because they all share the same position the last element in the HTML document will be the visible .box. The blue box, or .box:nth-child(3) is the visible box.

By assigning z-index: 1 to the .box:nth-child(1) we change the layer order of the .box and the first HTML element is the visible box. The green .box is now on top of the blue .box.

By assigning a higher z-index to the .box:nth-child(2) the red .box is now has the highest layer order and is the visible box.
